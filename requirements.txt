# Notes:
# 1. llama-cpp-python on Windows: the plain `pip install llama-cpp-python` gives you a
#    CPU-only build. For GPU offload on your GTX 1650, install a CUDA prebuilt wheel instead:
#       pip uninstall llama-cpp-python
#       pip install llama-cpp-python --extra-index-url https://abetlen.github.io/llama-cpp-python/whl/cu121
#    (pick the cuXXX tag matching your installed CUDA toolkit). Then server.py's
#    N_GPU_LAYERS setting will actually offload layers to the GPU.
# 2. Embeddings use plain `transformers` (AutoTokenizer + AutoModel with manual
#    mean-pooling) instead of sentence-transformers -- no extra wrapper package.
#    A CPU-only torch install is fine; the embedding model is tiny (MiniLM-L6,
#    ~90MB) and runs fast on CPU.
